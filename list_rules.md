# 銘柄リストアップ（フィルタリング）論理仕様

オニールの **CAN-SLIM** とDUKE。の **新高値ブレイク投資術** を統合したルール。

---

## 1. 必須通過条件 (Hard Filters)

以下をすべて満たさない銘柄は即除外。

### 1-1. 新高値ブレイク（N / DUKE基本条件）
- 52週高値（年初来高値）または上場来高値を更新していること
- **正しいベースからのブレイク**であること（単なる逆張りの高値掴みでない）
  - カップウィズハンドル、フラットベース、VCP（ボラティリティ収束）などのベースパターンを確認

### 1-2. 出来高スパイク（N・S・DUKE共通）
- ブレイク当日の出来高が直近25日平均の **150%以上**
- できれば **200%以上**（確信度の高いブレイク）
- 出来高を伴わない新高値は「だまし」の可能性大 → 除外またはWATCH格下げ

### 1-3. 最低流動性（S）
- 1日の平均売買代金 **5億円以上**（即座に売買成立できる流動性）

### 1-4. 低位株・万年割安株の除外
- 株価 **100円以上**（仕手性の強い低位株を排除）
- 新高値を伴わない低PBR放置株は除外（バリュートラップ）

---

## 2. 成長性スクリーニング (CAN-SLIM: C・A)

### 2-1. C：当期業績の加速（Current Quarterly Earnings）
最優先で確認する。**売上とEPSの両方が伸びていること**が必要。

| 指標 | 最低基準 | 理想 |
|---|---|---|
| 当期EPS成長率（前期比） | **+25%以上** | +40〜100%以上 |
| 当期売上高成長率（前期比） | **+25%以上** | 3四半期連続で加速 |
| EPS加速 | 直近期の成長率 > 前期の成長率 | 2〜3期連続で加速 |

- 売上を伴わないEPS増加（経費削減のみ）は持続性がなく評価しない
- 前期比が単調増加（加速トレンド）している銘柄を最優先

### 2-2. A：年間業績の安定的成長（Annual Earnings Increases）
過去3期以上の年次業績で確認する。

| 指標 | 基準 |
|---|---|
| 年間EPS成長率 | **3年連続+25%以上**（最低でも直近2期） |
| ROE | **17%以上**（オニール基準）、できれば20%超 |
| 営業利益率 | 改善傾向にあること |

- 1期でも赤字や大幅減益があった場合は慎重に判断
- EPS成長が「加速」している（直近期の成長率が高まっている）銘柄を優遇

---

## 3. 市場での優位性（CAN-SLIM: L・I）

### 3-1. L：主導銘柄か（Leader or Laggard）
- **過去3ヶ月・6ヶ月の騰落率がTOPIXを上回っている**こと
- 「市場が下落している日に逆行高している」銘柄は最優先
- RS（レラティブストレングス）は上位30%以内が理想（IBD基準 RS70以上相当）

### 3-2. I：機関投資家の動向（Institutional Sponsorship）
- 直近決算で機関投資家の保有が増加傾向にあること
- 信用倍率が低い（売り圧力が少ない）銘柄を優遇

---

## 4. ビッグチェンジの確認（N・DUKE）

新高値を更新する銘柄には必ず「変化」がある。以下を確認する。

- **新製品・新サービス・新業態**の展開（売上が急増する構造変化）
- **新経営陣**による大変革（ターンアラウンド系）
- **業界トレンドの追い風**（テーマ株・政策株）
- **上方修正・増配・自社株買い**の発表（大型株で特に重要）

AI判定時にウェブ検索で最新ニュースを確認すること。

---

## 5. 除外条件

以下に該当する銘柄は原則除外。

| 条件 | 理由 |
|---|---|
| 株価100円未満 | 仕手性が強く予測困難 |
| 売買代金5億円未満 | 流動性リスク・脱出困難 |
| 出来高スパイクなしの新高値 | だましの可能性 |
| 年間EPSが2期以上連続減少 | 成長トレンド崩壊 |
| 万年割安株（新高値を伴わない低PBR） | バリュートラップ |
| 市場全体が下落トレンド（M条件NG） | 逆風の中では勝率低下 |

---

## 6. スコアリング基準（加点方式）

各条件の達成度でスコアリングし、高スコア銘柄を監視リスト優先候補とする。

| 条件 | 加点 |
|---|---|
| 新高値更新 | +20 |
| 株価100円以上 | +10 |
| 売買代金5億円以上 | +30 |
| 売買代金20億円以上 | +5（追加） |
| 売買代金100億円以上 | +10（追加） |
| 出来高スパイク 150%以上 | +20 |
| 出来高スパイク 300%以上 | +10（追加） |
| EPS成長率 25%以上（C条件） | +10 |
| EPS加速（A・C条件） | +8 |
| 売上成長率 25%以上（C条件） | +10 |
| 売上加速 | +5 |
| 上方修正あり | +10 |
| ROE 17%以上（A条件） | +5 |
| ROE 20%以上 | +3（追加） |

> **最優先銘柄**: 「新高値更新」＋「売上/EPS成長率+25%以上」＋「出来高スパイク」がすべて揃い、かつビッグチェンジが確認できる銘柄。
