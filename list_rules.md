# 銘柄リストアップ（フィルタリング）論理仕様

オニールの **CAN-SLIM** と DUKE の **新高値ブレイク投資術** を統合したルール。

---

## 1. 必須通過条件 (Hard Filters)

以下をすべて満たさない銘柄は **passed=false** となる。
ただし失敗しても後続のスコアリングは継続して実施する（参考スコアとして表示）。

| # | 条件 | 基準 | 失敗時のメッセージ例 |
|---|------|------|---------------------|
| 1 | 新高値更新 | 52週高値・上場来高値の更新 | ❌ 新高値更新なし |
| 2 | 株価 | 100円以上 | ❌ 株価XX円 - 低位株除外（100円未満） |
| 3 | 売買代金 | 5億円以上 | ❌ 売買代金X.X億円 - 5億円未満（流動性不足） |
| 4 | 出来高スパイク | 25日平均比150%以上（データなし時は通過） | ❌ 出来高比率XX% - 25日平均の150%未満（スパイクなし） |
| 5 | 年次EPS | 2期連続マイナス → 除外 | ❌ 年間EPS2期連続減少（X%, X%）- 成長トレンド崩壊 |

---

## 2. スコアリング基準（100点満点）

CAN-SLIM の優先度に従い配点。

### C条件：当期業績（最大45pt） ← 最重要

| 指標 | 基準 | 加点 |
|------|------|------|
| EPS成長率 | 100%以上 | +20pt |
| EPS成長率 | 50〜99% | +15pt |
| EPS成長率 | 25〜49% | +10pt |
| EPS成長率 | 0〜24%（増益） | +3pt |
| EPS加速 | 直近期の成長率 > 前期 | +5pt |
| 売上成長率 | 25%以上 | +8pt |
| 売上成長率 | 0〜24%（増収） | +3pt |
| 売上加速 | 直近期の伸び > 前期 | +4pt |
| 上方修正 | 直近で上方修正あり | +8pt（N条件） |

### A条件：年間業績（最大25pt）

| 指標 | 基準 | 加点 |
|------|------|------|
| 年次EPS成長率 | 直近3期すべて+25%以上 | +12pt |
| 年次EPS成長率 | 直近2期+25%以上 | +6pt |
| ROE | 20%以上 | +8pt |
| ROE | 17〜19%（オニール基準） | +5pt |
| 営業利益率 | 改善傾向 | +5pt |

### L条件：主導銘柄（最大18pt）

TOPIX比相対強度（RS）で判定。

| 基準 | 加点 |
|------|------|
| 3M・6M両方 +15%超（真の主導銘柄） | +18pt |
| 3M・6M両方プラス | +12pt |
| 3M・6M片方がプラス | +6pt |

### S条件：出来高強度ボーナス（最大7pt）

150%は必須条件として通過済み。200%超から加点。

| 基準 | 加点 |
|------|------|
| 500%超（強烈スパイク） | +7pt |
| 300〜499% | +5pt |
| 200〜299% | +3pt |

### I条件：機関投資家動向（最大5pt）

| 基準 | 加点 |
|------|------|
| 直近6ヶ月以内に5%超保有者の増加報告あり | +3pt |
| 信用倍率 1.5倍未満（売り圧力少） | +2pt |

---

## 3. スコアの読み方

| スコア | 評価 |
|--------|------|
| 70〜100pt | 高優先度（全条件が揃っている） |
| 40〜69pt | 中優先度（一部条件不足） |
| 0〜39pt | 低優先度（必須条件は通過したが業績・RS不足） |
| passed=false | 必須条件未通過（参考スコアのみ） |

---

## 4. 除外対象

| 対象 | 理由 |
|------|------|
| ETF・商品ファンド | 業績テーブルなし → hasBizData=false で自動除外 |
| 株価100円未満 | 仕手性が強く予測困難 |
| 売買代金5億円未満 | 流動性不足・脱出困難 |
| 出来高スパイクなし | だまし高値の可能性 |
| 年次EPS2期連続マイナス | 成長トレンド崩壊 |

---

## 5. フィルター結果の表示仕様（詳細画面）

| 表示記号 | 意味 |
|----------|------|
| ❌ ... | 必須条件失敗 |
| ✅ ... | 必須条件クリア |
| ✓ ...pt | 加点項目（緑） |
| ! ... | 警告・減速・注意 |
| （テキストのみ） | 情報（売買代金・出来高比率等） |
| ? ... | データ未取得 |
