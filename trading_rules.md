# AI売買判定・シグナル生成ルール

## 1. 買い（エントリー）判定

AIはチャート画像から以下の「パワーの溜まり具合」を解析する。

- **ベース形成の質:** - **大型株:** 押し浅い横ばい（フラットベース）や、25日線へのタッチからの反発。
  - **中小型株:** カップウィズハンドルや、VCP（ボラティリティ収束）パターン。
- **ピボット突破の強度:** - 抵抗線を抜ける際、日中足で「大きな買い板」を飲み込むような強い動きがあるか。
- **判定ロジック:** - 抵抗線突破直後 ➔ **[BUY]**
  - 5%以上の乖離 ➔ **[WATCH]** (押し目待ち)

## 2. 売り（エグジット）判定

時価総額により「ボラティリティ」の許容範囲を変える。

- **損切り:** - 基本 **-7%**。ただし、超大型株でボラが低い場合は **-5%** などタイトに設定。
- **利確（トレンドフォロー）:**
  - 終値で **25日移動平均線** を完全に割り込むまでホールド（大型株の長期トレンドを逃さないため）。
- **異常値（クライマックス）:** - 週足レベルで数週間乖離しすぎた場合、出来高の異常増を伴う局面で半分利確を指示。

## 3. 監視継続

- **「新高値圏でのもみ合い」**は、さらなる上昇への準備期間と捉え、安易に手放さない。
